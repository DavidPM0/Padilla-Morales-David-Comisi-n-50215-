Proyecto: Aplicación de Gestión de Blog Personal (Comisión 50215)

Autor: David Padilla Morales
Fecha: 24/03/2024
Versión: 1.0

Objetivo: El objetivo del proyecto es permitir la gestión de un blog personal donde se logre administrar el contenido de los blogs, proyectos, eventos y merchandising.  

Consideraciones: 
instalar python -m pip install Pillow

Modelos usados:
- Blog: Es una seccion de blog personal.  
- Proyectos: Es una seccion relacionado a los proyectos realizados.
- Eventos TI: Se trata de futuros eventos relacionados a la tecnologia.
- Merchandising: Son productos que se intenta vender para poder obtener algunos ingresos.

Detalle:
La pagína muestra en el menu principal inicio, acerca de mí, blog, proyectos, eventos ti, merchandising, login, registrarse pero solo es para que el usuario pueda ver el contenido ya que si te logeas tienes acceso a las paginas de administracion de los modelos.
